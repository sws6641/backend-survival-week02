# 백엔드 생존코스 2주차 과제

## REST API 설계

### 예시

- 게시글 조회 `GET /posts`
- 게시글 작성 `POST /posts`
- 게시글 수정 `PATCH /posts/{id}`
- 게시글 삭제 `DELETE /posts/{id}`

### 로그인/회원가입

- 로그인
- 로그아웃
- 회원가입

### 내 정보

- 내 정보 
- 내 정보 수정 

### 상품

- 상품 목록
- 상품 상세

### 상품 리뷰

- 상품에 리뷰 작성
- 상품에 리뷰 수정
- 상품에 리뷰 삭제

### 장바구니

- 장바구니에 상품 추가
- 장바구니에 상품 삭제
- 장바구니 (담긴 상품 목록)

### 주문하기

- 주문 목록
